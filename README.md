# Deskripsi Analisis 

Proyek ini bertujuan untuk menganalisis dataset restoran dari platform Zomato menggunakan bahasa pemrograman Python. Analisis dilakukan untuk memahami tren restoran, preferensi pelanggan, serta hubungan antara jenis restoran, layanan pemesanan online, rating, dan kisaran harga.

# Dataset 

Dataset yang digunakan terdiri dari 148 baris data dan 7 kolom utama:
- name = Nama Restoran 
- online_order = ketersediaan pemesanan online
- book table = ketersediaan pemesanan meja
- rate = rating restoran
- votes = jumlah penilaian
- approx_cost (for two people) = perkiraan biaya untuk dua orang
- listed_in(type) = Jenis Restoran

# Tools dan Library 

- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn

# Tahapan Analisis 

1. Membaca dan memeriksa struktur dataset.
2. Pembersihan data dan konversi tipe data.
3. Analisis jenis restoran dan popularitas berdasarkan jumlah votes.
4. Analisis ketersediaan layanan pemesanan online.
5. Analisis distribusi rating restoran.
6. Analisis kisaran harga untuk dua orang.
7. Perbandingan rating berdasarkan mode pemesanan online dan offline.

# Hasil Utama 

- Mayoritas restoran telah menyediakan layanan pemesanan online.
- Restoran dengan kisaran harga menengah lebih diminati pelanggan. 
- Sebagian besar restoran memiliki rating pada rentang menengah hingga tinggi. 
- Layanan pemesanan online cenderung berkolerasi dengan tingkat kepuasaan pelanggan yang lebih baik.

# Kesimpulan 

Analisis menunjukkan bahwa digitalisasi layanan, harga yang sesuai, dan kualitas layanan merupakan faktor penting dalam meningkatkan kepuasan pelanggan dan daya saing restoran.
